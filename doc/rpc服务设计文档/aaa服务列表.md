



## Rpc服务划分

| 服务         | 说明                      |
| ------------ | ------------------------- |
| auth         | 注册登录服务              |
| user         | 用户信息的查询和更新服务  |
| msg          | 数据收发服务              |
| conversation | 服务端会话列表服务        |
| group        | 群组服务                  |
| push         | 推送服务                  |
| gateway      | 网关服务                  |
| ws           | 长连接服务，使用websocket |
